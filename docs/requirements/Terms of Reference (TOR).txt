Terms of Reference (TOR)
โครงการพัฒนาระบบ Adaptive Data Reliability & Lineage Platform (ADRLP)
ชื่อโครงการ

ระบบตรวจสอบคุณภาพ ติดตามเส้นทาง และบริหารความน่าเชื่อถือของข้อมูลอัตโนมัติสำหรับองค์กร

1. หลักการและเหตุผล

องค์กรจำนวนมากมีข้อมูลกระจายอยู่หลายระบบ เช่น ฐานข้อมูล ระบบเว็บไซต์ API และไฟล์ข้อมูล ส่งผลให้เกิดปัญหา

ข้อมูลผิดพลาด
Schema เปลี่ยนโดยไม่รู้ตัว
ข้อมูลซ้ำ
Dashboard แสดงผลผิด
ใช้เวลาตรวจสอบนาน

ในหลายกรณีปัญหาไม่ได้เกิดจากการวิเคราะห์ แต่เกิดจากการที่ข้อมูลไม่มีความน่าเชื่อถือ

โครงการนี้จึงพัฒนาระบบกลางสำหรับ

ประมวลผล ETL

แจ้งเตือนความผิดปกติ

โดยใช้ทรัพยากรฟรีและฐานข้อมูลของมหาวิทยาลัย
